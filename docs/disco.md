# Disco

Disco is the service discovery and name resolution system for Offline Lab OS. It lets nodes find each other by name without DNS servers, manual configuration, or internet.

Source: [github.com/offline-lab/disco](https://github.com/offline-lab/disco) · Docs: [disco.offline-lab.com](https://disco.offline-lab.com)

See the [Discovery](discovery/) section for the full documentation: protocol spec, CLI reference, and configuration.
