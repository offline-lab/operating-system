OLPF

The Offline Lab Package Format. OLPF defines how services are packaged, distributed, and run on MoreOS — a squashfs image containing everything the service needs, wrapped in a systemd portable service. No package managers, no dependency resolution.

Packages are self-contained and can be distributed over the internet, copied to a USB drive, or transferred over the local network.
